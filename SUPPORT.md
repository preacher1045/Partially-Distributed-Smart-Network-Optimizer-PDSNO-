# Support

Thanks for using PDSNO.

## Where To Get Help

For usage questions, bug reports, and feature requests, use the repository
GitHub Issues page:

- [GitHub Issues](https://github.com/AtlasIris/PDSNO/issues)

Please include:

- PDSNO version or commit hash
- OS and Python version
- Steps to reproduce
- Expected behavior
- Actual behavior
- Relevant logs or tracebacks

## Security Issues

Do not report security vulnerabilities in public issues.

Follow the process documented in SECURITY.md.

## Before Opening An Issue

1. Read README.md and docs/meta/01_INDEX.md.
2. Check existing open and closed issues.
3. Prepare a minimal reproducible example.

## Scope And Response Expectations

Support model and response expectations are described in:
